SIXBOOKS – Online Bookstore
1. Giới thiệu
Web bán sách online fullstack, cho phép người dùng xem sách, cập nhật thông tin cá nhân, upload ảnh và đặt hàng.
2. Công nghệ
Frontend: Angular, TypeScript, Bootstrap, Angular Material
Backend: Node.js, Express
Database: MongoDB (Mongoose)
Khác: Multer (upload ảnh), Toastr (thông báo)
3. Tính năng chính
Đăng ký / đăng nhập
Cập nhật profile + upload avatar
Xem danh sách & chi tiết sách
Thêm vào giỏ hàng, tạo đơn hàng
3. Quản lý sản phẩm (CRUD)
💡 Điểm nổi bật
Xây dựng fullstack từ đầu (FE + BE + DB)
Thiết kế REST API
Upload & hiển thị ảnh qua API
Giao diện responsive
4. Hạn chế
Ảnh lưu local (có thể mất khi deploy)
Chưa tối ưu performance
5. Tổng kết
Đây là một dự án nhóm mà chúng tôi thực hiện từ khi còn là sinh viên, với sự tham gia của 3 thành viên. Hệ thống ban đầu chủ yếu tập trung vào quản lý với
các tính năng CRUD cơ bản, phục vụ thao tác dữ liệu hiệu quả.
Hiện tại, hệ thống đã được tôi cải thiện giao diện người dùng (UI), mang đến trải nghiệm trực quan và bắt mắt hơn. Ngoài ra, tôi đã thêm các thông báo toast để 
người dùng nhận phản hồi tức thì, đồng thời tối ưu và cải thiện code theo các chuẩn hiện đại, giúp dự án dễ bảo trì và thuận tiện mở rộng trong tương lai.
