# SIXBOOKS – Online Bookstore

1.Mô tả
Hệ thống bán sách online fullstack, cho phép người dùng xem sách, quản lý thông tin cá nhân, upload ảnh và đặt hàng.
2.Công nghệ
- **Frontend:** Angular, TypeScript, Bootstrap, Angular Material  
- **Backend:** Node.js, Express  
- **Database:** MongoDB (Mongoose)  
- **Khác:** Multer (upload ảnh), Toastr (thông báo)
3.Chức năng chính
- Đăng ký / đăng nhập người dùng  
- Quản lý profile & upload avatar  
- Xem danh sách & chi tiết sách  
- Thêm sách vào giỏ hàng & tạo đơn hàng  
- Quản lý sản phẩm (CRUD)  
- Hệ thống voucher và soft delete  
4. Điểm nổi bật
- Xây dựng **fullstack từ đầu** với 75+ RESTful APIs  
- Sử dụng **MongoDB Aggregation** để join nhiều collection, tối ưu dữ liệu  
- Full-text search & server-side pagination  
- Upload & hiển thị ảnh qua API  
- Giao diện **responsive**, thông báo toast tức thì  
- Code được chuẩn hóa, dễ bảo trì & mở rộng  
5. Hạn chế
- Ảnh lưu local (có thể mất khi deploy)  
- Chưa tối ưu cache / indexing cho performance lớn  
6. Vai trò cá nhân
- Backend Developer: Thiết kế & triển khai APIs cho sách, đơn hàng, voucher  
- Tối ưu dữ liệu với Aggregation Pipelines  
- Hỗ trợ frontend chuẩn hóa API và DTOs  

**Nhóm:** 3 thành viên
